Описание проекта: Туристическое агентство «Sea & Mountains»

Данный проект представляет собой веб-приложение для туристического агентства, разработанное на фреймворке **Flask** (Python) с использованием **SQLite** в качестве базы данных и интеграцией платёжной системы **ЮKassa**. Приложение позволяет пользователям просматривать туры, добавлять их в корзину, оплачивать заказы через ЮKassa и отслеживать историю платежей в личном кабинете.

## Основной функционал

- **Главная страница** – отображение всех доступных туров с фильтрацией по категориям (море, горы, страны). Каждый тур содержит название, описание, цену и изображение.
    ![alt text](images\image.png)
- **Корзина** – добавление туров в корзину, автоматический подсчёт общей суммы.
    ![alt text](images\image(1).png)
- **Оплата** – интеграция с ЮKassa: создание платежа, перенаправление пользователя на защищённую страницу оплаты, обработка результата через вебхук.
    
- **Личный кабинет** – отображение баланса (виртуального), содержимого корзины и истории заказов с их статусами.
    ![alt text](images\image(2).png)
    ![alt text](images\image(3).png)
- **Вебхук** – асинхронное обновление статуса заказа при успешной оплате, автоматическая очистка корзины и начисление виртуальных средств на баланс пользователя.

## Модели данных

- **User** – пользователь (id, имя, баланс).
    
- **Tour** – тур (id, название, категория, цена, описание, путь к изображению).
    
- **CartItem** – связь пользователя и тура (корзина).
    
- **Order** – заказ (id, id пользователя, сумма, статус, payment_id от ЮKassa).
    

## Инструкция по запуску

1. **Установите зависимости**:
    
    bash
    
    pip install flask flask-sqlalchemy yookassa
    
2. **Настройте ЮKassa**:
    
    - Зарегистрируйтесь в [ЮKassa](https://yookassa.ru/).
        
    - Получите `shopId` и `secretKey` (тестовые ключи доступны в личном кабинете).
        
    - В файле `app.py` замените `'123456'` и `'test_...'` на свои ключи:
        
        python
        
        Configuration.configure('ваш_shopId', 'ваш_secretKey')
        
3. **Запустите приложение**:
    
    bash
    
    python app.py
    
    Сервер будет доступен по адресу `http://127.0.0.1:8010`.
    
4. **Тестовый пользователь**:
    
    - При первом запуске автоматически создаётся пользователь `Traveler` с балансом `0.0` и 6 тестовых туров.
        
    - Для оплаты используйте тестовые карты ЮKassa (например, `5555 5555 5555 4477` с любым сроком и CVC).
        

## Особенности реализации

- **Статическая раздача изображений** – маршрут `/images/<filename>` обслуживает картинки из папки `images`.
    
- **Генерация уникального ключа идемпотентности** для каждого платежа (UUID).
    
- **Вебхук** (`/webhook`) принимает уведомления от ЮKassa и обновляет статус заказа. Для продакшена требуется публичный URL (через ngrok или хостинг).
    
- **Баланс пользователя** увеличивается на сумму оплаченного заказа (демонстрационная логика, может быть легко заменена на реальную).
